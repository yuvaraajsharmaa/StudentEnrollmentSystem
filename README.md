# Student Enrollment System 

A Java-based Student Enrollment System tha tmanages students, courses, and enrollments. 

## Files 
- Student.java: Defines the Student class
- Course.java : Defines the Course class
- Enrollment.java: Handles enrollment logic
- Main.java: Entry point of the application

## How to Run 
1. Compile: javac *.java
2. Run: java Main
